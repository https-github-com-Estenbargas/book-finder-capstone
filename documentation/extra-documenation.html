Goals

  • I want to create a website which lets people both search books and save that book.

  • A Preset Category which uses filters to narrow user searches.
  • Sorting system to mark book if read or not.
  • Preview for book on the page.
  • Simple, easy to use interface.
  • A way to add your own books to your library and possibly have the ability to share to others.
